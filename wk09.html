BootStrap

From this lesson, I learnt about:


  1. Introduction to Bootstrap:
    • Bootstrap is a framework developed by bored developers to simplify UI development and address browser and device compatibility issues.
  2. Bootstrap Features:
    • User interface libraries and utilities are included in Bootstrap.
    • Utilizes Flexbox grid for layout, rem for measurements, and device breakpoints for different media sizes.
  3. Bootstrap Grid System:
    • Grid divided into 12 columns for creating responsive layouts.
    • Columns specified using classes like col-sm-2 for small devices.
    • If no size specified, Bootstrap auto-fills the column size.
    • To use the grid system, add a .row class to the main div.
  4. Documentation Reference:
    • Good to practice referring to the official Bootstrap documentation for detailed examples and explanations.
