Introduction to Front-End Development

From this lesson, I learnt about:


  • What web development is all about
  • Roles and Responsiblities of a web developer
  • Web production workflow
  • Structure of a HTML file documment
  • HTML element: The root element of an HTML page
  • Head element: Contains meta information about the HTML page
  • Body element: Defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • h1-6 element: HTML headings are titles or subtitles that you want to display on a webpage.
  • img tag: Used to embed an image in a web page from other web pages or a local file
  • ul & ol tags: Allow web developers to group a set of related items in lists.
  • p element: Defines a paragraph block element
  • em tag: This tag is used to define emphasized text. The content inside is typically displayed in italic. A screen reader will pronounce the words in the tag with an emphasis, using verbal stress.
  • a tag: The tag defines a hyperlink.
  • div tag: The tag defines a division or a section in an HTML document.
  • span tag: The tag is an inline container used to mark up a part of a text, or a part of a document.
