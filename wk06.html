Javascript Objects

From this lesson, I learnt about:


Definition: Objects group together a set of variables and functions to create a model

  1. Literal Objects & Function-Based Objects:
    • Literal objects are created using curly braces {} and can contain properties and methods.
    • Function-based objects are created using constructor functions, allowing the creation of multiple instances with shared properties and methods.
  2. Variable Declaration & Hoisting:
    • Variables declared using var are hoisted to the top of the function or global scope.
    • The scope of a variable defined with var is function scope or global if declared outside any function.
  3. LET Declarations:
    • let declarations are not hoisted to the top of the enclosing block.
    • let is block-scoped, meaning it only exists within the curly braces it is contained in.
  4. VAR vs LET:
    • var allows re-declaration of a variable, whereas let does not.
  5. LET: NO REDECLARATION:
    • If an identifier has already been defined in a scope, using the identifier in a let declaration inside that scope causes an error.
    • let will not redefine another variable in the same scope.
  6. CONST Declarations:
    • Variables declared using const are constants and cannot be changed once set.
    • Every const variable must be initialized on declaration.
  7. Objects with CONST:
    • const prevents modification of the binding (variable reference), but not of the value itself.
    • Changing properties of an object declared with const is allowed.
  8. For...of Loop:
    • The for...of loop is used to iterate over iterable objects like arrays, maps, sets, strings, etc.
    • It simplifies the syntax for iterating over elements in an iterable.
