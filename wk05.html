JavaScript Basic

From this lesson, I learnt about:


  • How to create a Variable: var x; OR let y;
  • How to use Variables:
    x = 5;
    y = 6;
    let z = x + y;
  • Strings:
    1. A JavaScript string is zero or more characters written inside quotes
    2. You can use single or double quotes
    3. The backslash (\) escape character turns special characters into string characters
    4. To find the length of a string, use the built-in length property (Eg. var.length)

    Visit for string properties and methods: https://www.w3schools.com/js/js_string_methods.asp
  • Numbers:
    1. JavaScript has only one type of number. Numbers can be written with or without decimals.
    2. Extra large or extra small numbers can be written with scientific (exponent) notation
    Eg.
    let x = 123e5; // 12300000
    let y = 123e-5; // 0.00123
    3. Integers (numbers without a period or exponent notation) are accurate up to 15 digits
    4. The maximum number of decimals is 17
    5. Infinity (or -Infinity) is the value JavaScript will return if you calculate a number outside the largest possible number
    6. NaN is a JavaScript reserved word indicating that a number is not a legal number

    More about Numbers: https://www.w3schools.com/js/js_math.asp
    More about Math Object: https://www.w3schools.com/js/js_math.asp
  • Boolean Values:
    JavaScript has a Boolean data type. It can only take the values "true" or "false"
    More about Boolean and its operators: https://www.w3schools.com/js/js_booleans.asp
  • Const:
    1. Variables defined with const cannot be Redeclared
    2. Variables defined with const cannot be Reassigned
    3. Variables defined with const have Block Scope
    More about const: https://www.w3schools.com/js/js_const.asp
  • Objects:
    1. Objects are variables too. But objects can contain many values
    2. The values are written as name:value pairs (name and value separated by a colon).
    Eg. person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    3. You can access objects in two ways:
    objectName.propertyName
    OR
    objectName["propertyName"]

    More about Objects: https://www.w3schools.com/js/js_objects.asp
  • Arrays: Basically the same thing as lists in python
    More about arrays: https://www.w3schools.com/js/js_arrays.asp
  • Comments: Use "//" for one-line comments and "/* ... */" for multiple-line comments
  • Regular Expressions: https://www.w3schools.com/js/js_regexp.asp
  • Comparison Operators: https://www.w3schools.com/js/js_comparisons.asp
  • Arithmetic: https://www.w3schools.com/js/js_arithmetic.asp
  • Logic Operators:
    1. && operator means 'and' (Second Precedence)
    2. || operator means 'or' (Last precedence)
    3. ! operator means 'not' (Highest precedence)
  • if else conditions: https://www.w3schools.com/js/js_if_else.asp
  • switch statement: https://www.w3schools.com/js/js_switch.asp
  • if else one-liners:
    if (cherub === 'Cupid') console.log("Ouch, an arrow! Ooo, I'm in love!")
    else console.log("I feel nothing");
  • if else Ternaries:
    var job = (animal === 'cat') ? 'cat herder' : 'dog catcher';
  • typeof: https://www.w3schools.com/js/js_typeof.asp
  • for loops (Sequential): https://www.w3schools.com/js/js_loop_for.asp
  • for loops (Enumerative): https://www.w3schools.com/js/js_loop_forin.asp
  • sets: https://www.w3schools.com/js/js_sets.asp
  • maps: https://www.w3schools.com/js/js_maps.asp
  • while loops: https://www.w3schools.com/js/js_loop_while.asp
